# 📘 Assignment: Testing and Debugging

## 🎯 Objective

Learn how to write tests and find bugs in Python code by using assertions, verifying function behavior, and correcting broken logic.

## 📝 Tasks

### 🛠️ Write Unit Tests for Validation Functions

#### Description
Complete the functions in `starter-code.py` and add assertions to test each function. Use these tests to confirm that valid inputs work and invalid inputs are handled correctly.

#### Requirements
Completed program should:

- Implement `is_prime(number)`, `calculate_average(numbers)`, and `validate_password(password)`.
- Use `assert` statements in the `run_tests()` function to verify expected results.
- Include tests for valid cases and edge cases, such as empty lists or invalid passwords.

### 🛠️ Debug and Fix a Broken Function

#### Description
Identify the bug in the `divide_numbers(a, b)` function and fix it. Add assertion checks to make sure the function behaves safely for all inputs.

#### Requirements
Completed program should:

- Correctly divide two numbers and return the result.
- Protect against division by zero with an assertion.
- Demonstrate the fixed function with example inputs and expected outputs.
